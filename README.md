Project documentation for porting the current Green Leaf Pacific OpenCart storefront to the new React + ERPNext stack.

Keep technical notes, migration decisions, audit results, and implementation plans for making the new storefront visually match the existing OpenCart site while using the newer ERPNext-backed architecture.
